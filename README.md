# Spryker Migration Analyser

## Installation & Usage

1. Fork/Clone/Download this repo.
2. Copy/Paste your `composer.json` and `composer.lock` contents into one of the existing files inside `composer/` folder
3. Change inside `page.html` the properties that holds your data
4. Open `page.html` inside your browser and enjoy
